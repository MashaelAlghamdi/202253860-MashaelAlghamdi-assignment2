# Assignment 2 – Foundation & AI Integration

## Project Description

This project is a responsive personal portfolio website developed using HTML, CSS, and JavaScript.  
It serves as the foundation for my professional portfolio and demonstrates core front-end development concepts including semantic structure, responsive layout design, and basic interactivity.

The website includes an About section, a Projects section displaying project cards, and a Contact form with JavaScript validation. Several interactive features were implemented to improve usability, including project search and sorting, expandable project descriptions, and a dark/light theme toggle with persistence using `localStorage`.

The contact form includes validation that checks for empty fields and verifies the email format, displaying an error message if the input is invalid. The project search feature also provides user feedback by displaying a message when no matching projects are found.

Additionally, the site includes a small dynamic feature that fetches a random programming fact from the Wikipedia API to demonstrate basic data handling using JavaScript.

You can open the project using one of the following methods.

**Option 1 – Open directly in your browser**

Open the `index.html` file in your browser.

**Option 2 – Using VS Code Live Server**

* Open the project folder in **Visual Studio Code**
* Install the **Live Server** extension
* Right-click `index.html`

## Short Summary of AI Use

AI tools such as **ChatGPT** and **GitHub Copilot** were used as development assistants during this project. They helped with debugging JavaScript logic, understanding CSS behavior, and improving the structure of the code.

ChatGPT was mainly used to clarify concepts such as implementing dark mode using `localStorage`, handling DOM events, and organizing the project structure. GitHub Copilot assisted by suggesting small code completions and helping with repetitive coding patterns.

All AI-generated suggestions were reviewed and modified when necessary to ensure they matched the project requirements and that the functionality was fully understood before being included in the final implementation.
